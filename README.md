# PracticalMachineLearning
Assignment For Coursera Practical Machine Learning course

##Description

This repo contains the course work for the Coursera Practical Machine Learning Assignment. Within this repo you will find the following files:

* R markdown (.Rmd) file of the assignment write up 

* html file of the assignment write up, this has been generated from the orignal R markdown file using knitr

* Data folder, this folder contains the orignal training and testing datasets for the assignment which were orignally downloaded from here: https://d396qusza40orc.cloudfront.net/predmachlearn/pml-training.csv and https://d396qusza40orc.cloudfront.net/predmachlearn/pml-testing.csv 

For more informaiton about the datasets used in this assignment please see here:http://groupware.les.inf.puc-rio.br/har
